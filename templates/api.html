RESTful API

This is a very simple API I constructed to serve some JSON. There really isn't much on the backend here, I just wanted to demonstrate a simple RESTful service that I built from scratch. If needed, we could do any amount of heavy lifting on the back end, crunching numbers, SQL queries and joins, or whatever else would fill a need. For ease of use, I'm not using any authentication here, so you can just test the API calls with curl, or just use your browser.

Weather:

To check your local weather, you can just curl "localhost:5000/api/weather/'your zip code here'. Conversely, you can just click my example link localhost:5000/api/weather/77573 to get an example response in the browser. You will get a JSON response for zip code you request.
